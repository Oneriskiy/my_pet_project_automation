## Описание проекта
### CLI-скрипт для управления заметками: добавление и удаление задач прямо из консоли. Позволяет удобно автоматизировать личные или рабочие задачи.

> ! все задачи сохраняются в папку `my_tasks` >  **tasks.txt**, логи в **logs.log**

---

## Установка 

### Созданием и активируем виртуальное окружение
`python -m venv .venv`
`.venv\Scripts\activate - Windows` 
`source .venv/bin/activate - Mac/Linux`
### при необходимости импортируем инструменты разработки 
`pip install -r requirements.txt`
### запускаем 
`python main.py`
___

>После этого проект полностью готов к использованию.
> > ⚠️ Обратите внимание: путь к рабочему столу может отличаться на вашем компьютере.  
Скрипт создаёт папку `my_tasks` в домашней директории пользователя.
